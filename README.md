# Anna University CGPA Calculator

![Anna University CGPA Calculator Logo](gpa.png)

A fast, student-friendly Anna University CGPA Calculator built for quick grade and credit based CGPA calculation.

[Live Demo](https://srirambharath-7.github.io/Anna-University-CGPA-Calculator/)

## Overview

This project helps Anna University students calculate CGPA instantly using subject grades and credits.
It was built as a lightweight backup tool that remains accessible when official university websites are slow,
under maintenance, or unavailable during result periods.

## Features

- Anna University grade-point calculation
- Dynamic subject rows
- Subject name, grade, and credit inputs
- Animated CGPA result card
- Mobile-friendly dark UI
- SEO-friendly static deployment on GitHub Pages

## Grade Points

| Grade | Point |
|-------|-------|
| S     | 10    |
| A+    | 9     |
| A     | 8     |
| B+    | 7     |
| B     | 6     |
| U     | 0     |

## Formula

CGPA = sum of (Grade Point x Credits) / sum of Credits
